HTML Terms

1. HTML comments are used by the programmer to explain their code to mostly other people. These codes are ignored by the browser.

2. ID attributes specify a certain id to an element

3. Class attributes specify a specific classname to an element

4. Block elements fill the entire space of its container

5. Inline elements do not start on a new line and they can take as much space as needed.

6. Grouping tags usually include tags like span and and div.

7. Inline Frames (iframes)are basically a HTML code that is embedded within another HTML document.

8. Meta tags provide us with data about the page, including the author, page description, keywords etc. This is not displayed by the browser.
